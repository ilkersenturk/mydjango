Selenium

WebDriver drives a browser natively, as a user would, either locally or on a remote machine using the Selenium server.

It marks a leap forward in terms of browser automation

First you need to install the Selenium bindings for your automation project.

The installation process for libraries depends on the language you choose to use.

Make sure you check the Selenium downloads page to make sure you are using the latest version.
